# react-basic

생활코딩 리액트 입문 수업

- 리액트를 사용하는 이유
- 리액트의 컴포넌트를 만드는 법
- 리액트와 불변(immutable)의 관계

- 리액트를 통해서 재사용 가능한 사용자 정의 태그(컴포넌트)를 만들기
- single page application을 구현하고 싶은 분
  <br><br>

# 목차

강의 순서는 다음과 같다.

1. 개발환경
2. 컴포넌트 제작
3. state
4. 이벤트
5. create 기능 구현
6. Update & Delete 기능 구현
